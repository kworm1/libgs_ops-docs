libgs_ops.scheduling.Communication¶

class libgs_ops.scheduling.Communication(cmd, retries=3, wait=True)[source]¶

This class holds the message (fully encoded) that is sent to the satellite.

It is just a convenience class that wraps and populates a dictionary properly for use by the other classes in this module.

Parameters:
  • cmd (string or bytearray) – Command string (fully encoded) to send
  • satellite. (to) –
  • retries (int) – Number of times the command should be retried in
  • of failure (case) –
  • wait (bool) – Whether the ground station should wait for a reply
  • the satellite or not. (from) –
