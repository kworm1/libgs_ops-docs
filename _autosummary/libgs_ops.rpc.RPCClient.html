libgs_ops.rpc.RPCClient¶

class libgs_ops.rpc.RPCClient(uri, timeout=60, transport=None, encoding=None, verbose=0, allow_none=0, use_datetime=0)[source]¶
An improvement on ye olde ServerProxy that
  • Times out if the connection has gone belly up
  • Allows kwargs to be passed along with arguments, by adding the kwargs as a single dictionary argument at the end of the list of parameters. If no kwargs are specified it will not be added, and therefore it is fully backwards compatible with normal SimpleXMLRPCServer.
